Cortex

Cortex is a versatile Discord bot that facilitates music playback and interaction management within a server. It allows users to join voice channels, play songs from YouTube, skip tracks, and leave channels. Songs are queued and played in sequence, with error handling for playback issues. The bot also enables administrators to change its activity status, including options like playing, listening, watching, or streaming, with support for stream URLs. Additionally, all actions, such as joining a voice channel, playing music, or skipping songs, are logged in a designated channel for transparency. Cortex is configured with secure environment variables and offers commands like help and test, ensuring a smooth user experience while maintaining control and visibility for admins.

Orbit

Orbit is a Discord bot with several management and logging features. It allows server administrators to set and manage channels for welcome, verification, and logs. It logs actions such as setting up channels, roles, and bot activity, and stores settings in a database. The bot also includes commands for modifying activity status, sending messages to specified channels, and verifying users based on specific roles. Error handling ensures smooth operation, and the bot can be customized for each server.
